You can use either a button with type="submit", or a button with type="button". The difference between these types is simply in how they behave when clicked. Both button types display as buttons and will cause the form data to be submitted to the server. The submit type has default behavior of submitting the form data. A button type has no default behavior, but can be driven by scripts which can tell it what to do. The JavaScript functions written are what drive the button type to submit the form when the button is clicked.
