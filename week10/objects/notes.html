Chapter 9 The Window Object

-Window object is the browser's global object, which represents the browser window that contains a web page

Browser Object Model

- a collection of properties and methods that contain info about browser and computer screen.

-We can find out which browser is being used to view a page, find ou the dimensions of screen it's viewed on, and which pages have been visited before the current page. Can also creeate pop-up windows (annoy your users! Ha!)

-No official standard for BOM, but there are a number of properties/methods supported by all major browsers making some sort of standards. They're available through the window object. Every browser window, tab, popup, frame, and iframe has a window object

Global

-Global variables are actual properties of a global object. Globals are created without const, let, or var keywords, and can be accessed in all parts of the program

In browser, the global object is the window object.

-parseInt() and isNaN() are methods of the global object or the window object. Full way to write is window.parseInt(4.2), but we usually shorten to parseInt() only.

Browser History

- Window history property can be used to access info about any previously visited pages in current browser session

- window.history.length property shows how many pages have been visited before the current page

- window.history.go() method can be used to go to a specific page (0 is current page)

- windowhistory.go(1); will go forward 1 page.
- window.history.go(0); reloads current page.
- window.history.go(-1); goes back 1 page.

- also can use window.history.forward() and window.history.back() methods

Screen Info

- window.screen object contains info about the screen the browser is displayed on. Find height and width of screen in pixels using height and width properties. (window.screen.height, and window.screen.width)

- availHeight and availWidth can be used to find height and width excluding operating system menus. window.screen.availWidth and window.screen.availHeight

- colorDepth can find the color bit depth of the user's monitor (mostly used for statistics) window.screen.colorDepth;

Document Object

- Each window contains document object. This has properties ad methods that deal with page that has been loaded into the window.

- document.write() method writes string of text to the page. If a page is already loaded it completely replaces current document

- You can add html as part of the document.write string.

- document.write() is not popular to use, highly frowned upon because it can only be used by mixing JS in an HTML document

Cookies

- Cookies are small files saved locally on user's computer.

- Browsers don't remember anything from one request to another, so every time a user visits a page, nothing is rmembered.

- cookies are used to sidestep the problem and store info that can be retrieved between requests.

- Cookies are restricted by only being read by a web page from the same domain that set them. This stops sites being able to access info about users from other sites they've visited.

- cookies are limited to storing up to 4kb of data, but 20 cookies are allowed per domain, so sometimes a lot of data

- can be used for personalizing user's browsing experience to store user prefereneces, keep tack of choices (like a shopping cart)

- to create a cookie, assign it to JavaScript "cookie jar" using document.cookie property

- document.cookie acts like a special type of string.

- To change a cookie value, reassign it to document.cookie using the same name with a different value.

Timing Functions

- setTimeout() method accepts a callback to a function as first parameter and number of milliseconds as its second parameter.

- clearTimeout() method cancels the timeout.

- setInterval() is similar to setTimeout but it repeatedly invokes callback function after those seconds.

Animation

- setTimeOut() and setInterval() methods can be used to animate elements on a web page.

Chapter 14 HTML5 APIs

HTML5

- W3C plans to develop future versions of HTML5 much more frequently than previously. HTML5.1 has already become latest and 5.2 is in development.

Data- Attribute

- data- attribute is a way of embedding data in a web page with custom attributes that are ignored by browser, and are private to a page

- Their sole purpose is to be used by JavaScript program. Perfect for adding data that can be used as hook to access info about particular element on page

- names of attributes must include format:

  • start with data-
  • contain only lowercase letters, numbers, hyphens, dots, colons, or underscores
  • include an optional string value

- each element has a dataset property that can be used to access any data- attribute

HTML5 APIs

- HTML5 specification contains a number of APIs that help gain access to hardware like cameras, batteries, geolocation and graphics card.

HTML5 Web Storage

- web storage API provides key-value store on client's computer similar to using cookies, but has more storage capapcity and is easier to use.

- good for storing info about users, and storing application-specific info to be used during future sessions

- different than cookies because info is not shared with server on every request, info is available in multiple windows of browser if domain is the same, and storage capacity limit is much greater than cookies.

- if a browser supports Web Storage API, window object will have property called localStorage. Also a sessionStorage object that works the same way, but stores data for only the current session

- instead of using getItem() and setItem() methods, you can assign them instead with localStorage.name, for example

- remove from local storage by using removeItem (localStorage.removeItem('name');

- delete local storage by using delete localStorage.name;

- clear everything or completely remove everything by using the clear() method-- localStorage.clear();

Geolocation

- Geolocation API used to obtain geographical postiion of device. It can find the exact location, and link to nearby places or measure speed at which user is moving. Crazy!

- example would be a search by location

- with privacy concerns, user must grant permission first!

- has several properties to find information about location and movement, like speed, altitude, heading, and timestamp.

Web Workers

- allow processes to be run in the background.

- use Worker() constructor function to create a new worker

- use postMessage() to send message and start worker working

- when worker has completed task, it can be stopped using terminate() method

Websockets

- new protocol that allows two-way communication with a server – also known as push messaging. This means that a connection is kept open and responses are ‘pushed’ to the client as soon as they are received.

Multimedia
